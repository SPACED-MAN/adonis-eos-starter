# Editor Quick Start

Welcome! This guide will help you get started creating and managing content in Adonis EOS.

## Logging In

1. Navigate to `/admin` on your site
2. Enter your email and password
3. You'll be taken to the Dashboard

## Your Dashboard

The dashboard shows all your content in one place:

- **Posts list**: All pages, blog posts, and other content
- **Search**: Find content by title or slug
- **Filters**: Filter by type, status, or locale
- **Create New**: Button to start a new post

## Creating Your First Post

1. Click **"Create New"** in the dashboard
2. Select a post type (Page, Blog, etc.)
3. Choose a language/locale
4. Fill in the basic information:
   - **Title**: The main heading for your content
   - **Slug**: The URL-friendly name (auto-generated from title)
   - **Excerpt**: Short summary (optional)

5. **Add Modules**: Click "Add Module" to add content blocks:
   - **Prose**: Rich text editor for paragraphs and formatting
   - **Hero Banner**: Large banner with image and call-to-action
   - **Gallery**: Image galleries
   - **Accordion**: Collapsible content sections
   - **CTA**: Call-to-action buttons
   - And more...

6. **Set Status**:
   - **Draft**: Work in progress, not visible publicly
   - **Review**: Ready for approval by Editor Admin
   - **Published**: Live on your website

7. Click **"Save Changes"** to save your work

## Publishing Workflow

Depending on your role, you'll have different publishing options:

### If you're an Editor
1. Create and edit your content
2. Click **"Save for Review"** when ready
3. An Editor Admin will review and approve your changes

### If you're an Editor Admin
1. Create, edit, and publish content directly
2. Click **"Save Changes"** or **"Publish Changes"**
3. Review and approve content from other editors

### If you're a Translator
1. Switch to your target language in the editor
2. Translate content and save changes
3. Submit for review when complete
