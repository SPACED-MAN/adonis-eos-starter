# For Developers

Technical documentation for developers and system administrators building with Adonis EOS.

Adonis EOS provides a robust, developer-first architecture built on AdonisJS 6 and React. Whether you're building custom modules, integrating AI agents, or configuring complex workflows, these guides will help you succeed.

## Sections

- **[Getting Started](./00a-getting-started.md)**: Installation and deployment.
- **[Concepts](./01-concepts.md)**: Core architecture and principles.
- **[Building Content](./02-content-management-overview.md)**: Understanding the content model.
- **[Automation](./06-automation.md)**: Workflows, Webhooks, and AI.

## Key Features

- **AdonisJS 6**: Modern, type-safe Node.js framework.
- **Inertia.js + React**: Seamless SPA experience with server-side routing.
- **AI-Native**: Built-in support for AI agents and Model Context Protocol (MCP).
- **Modular**: Highly extensible module and field systems.
