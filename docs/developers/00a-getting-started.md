# Getting Started

Learn how to install, configure, and deploy Adonis EOS.

Adonis EOS is designed to be developer-friendly, with a focus on quick setup and easy deployment. Whether you're starting a new project or integrating into an existing one, these guides will help you get up and running.

## In this section

- **[Installation](./00b-installation.md)**: System requirements and initial project setup.
- **[Deployment](./00c-deployment.md)**: Moving your application to production environments.

## Prerequisites

Before diving in, make sure you have:

- **Node.js 18+** (latest LTS recommended)
- **PostgreSQL 14+** (or another supported database)
- **npm** or **yarn**

If you're new to AdonisJS, we recommend checking out the [AdonisJS documentation](https://docs.adonisjs.com) to understand the core framework concepts.
