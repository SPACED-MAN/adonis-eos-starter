# Getting Started

Guides for installing, configuring, and deploying Adonis EOS to production.

Adonis EOS is built on AdonisJS 6 and React, providing a robust foundation for modern headless CMS development. Whether you're setting up a new project or migrating an existing one, these guides will help you get up and running quickly.

## In this section

- **[Installation](./01-getting-started/01-installation.md)**: System requirements and initial project configuration.
- **[Project Structure](./01-getting-started/02-project-structure.md)**: A deep dive into the folder organization and the `app/` directory.
- **[Deployment](./01-getting-started/03-deployment.md)**: Production build process, migrations, and process management.
- **[Update Philosophy](./01-getting-started/04-update-philosophy.md)**: Best practices for maintaining and upgrading your Adonis EOS installation.
